# ML Final Project

## Module 4 — Unsupervised analysis

PCA, K-Means, and DBSCAN are implemented from scratch with NumPy. To reproduce
the Module 4 metrics and figures on three bundled datasets, run from the project
root:

```bash
python src/experiments/run_all.py
```

Generated plots are saved under `figures/unsupervised/`. Run the tests with:

```bash
python -m pytest tests/test_unsupervised.py
```
